# HydroCarry Running Vest Review Summary (US)

## Aggregate Snapshot
- Average rating: 4.5 / 5
- Total reviews analyzed: 112
- Summary window: Last 12 months

## What Customers Like Most
1. Minimal bounce even at fast paces
2. Soft flasks are easy to drink from on the move
3. Tons of pocket space for nutrition and gear
4. Very lightweight for the capacity

## Common Complaints
1. Sizing is tricky; between sizes is common
2. Soft flasks develop a plastic taste if not cleaned regularly
3. Rear reservoir sleeve is tight with a full 1.5L bladder
4. Chest strap buckles can chafe without a shirt underneath

## Fit Theme Summary
- Adjustable but sizing between XS/S and M/L can be awkward
- Snug fit is intentional for bounce reduction
