# PowerBand Resistance Set

## Overview
Set of 5 fabric resistance bands in graduated resistance levels for warm-ups, strength training, and physical therapy exercises.

## Best For
- Warm-up and activation exercises
- Strength and glute training
- Travel-friendly workout kit
- Physical therapy and rehab

## Key Features
- 5 bands: Extra Light, Light, Medium, Heavy, Extra Heavy
- Fabric construction (non-roll, non-snap)
- Color-coded by resistance level
- Includes mesh carry bag
- Each band: 13" x 3" loop

## Materials
- Polyester / cotton blend with latex core
- Hand wash, air dry
