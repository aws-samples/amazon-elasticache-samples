# EnduroFit Compression Tight

## Overview
Full-length compression tight for running and recovery. Graduated compression supports circulation during and after activity.

## Best For
- Long-distance running
- Post-workout recovery wear
- Cold-weather base layer

## Key Features
- Graduated compression (firmer at calf, lighter at thigh)
- Flatlock seams to prevent chafing
- Wide waistband with internal drawcord
- Side pocket fits phone or gel
- Reflective calf detail

## Sizing
- Available: XS, S, M, L, XL, XXL
- Fit: Compression; size up if between sizes

## Materials
- 78% nylon / 22% elastane
- Machine wash cold, hang dry; do not use fabric softener
