# SunShield Running Cap

## Overview
Lightweight running cap with UPF 50+ protection, moisture-wicking sweatband, and laser-cut ventilation holes for hot-weather runs.

## Best For
- Running in sun and heat
- Outdoor training
- Everyday athletic wear

## Key Features
- UPF 50+ sun protection
- Laser-cut ventilation holes in crown
- Moisture-wicking internal sweatband
- Adjustable rear buckle closure
- Reflective logo
- Crushable and packable

## Sizing
- One size fits most; adjustable closure

## Materials
- 100% recycled polyester
- Machine wash cold, air dry
