# GearPack Running Backpack 15L

## Overview
Lightweight 15-liter running backpack for commute runs, long trail outings, and travel. Compression straps, hydration sleeve, and ventilated back panel.

## Best For
- Run commuting with a change of clothes
- Long trail runs requiring extra gear
- Travel and day hikes

## Key Features
- 15L main compartment with internal organizer
- Hydration reservoir sleeve (reservoir sold separately)
- Dual side water bottle pockets
- Compression straps to reduce bounce
- Ventilated mesh back panel
- Sternum strap and removable hip belt
- Reflective accents
- Weight: 12.5 oz

## Sizing
- One size; adjustable straps fit most torsos

## Materials
- Body: 210D ripstop nylon
- Back panel: 3D mesh
- Spot clean; air dry
