# HydroCarry Running Vest

## Overview
Lightweight hydration vest with dual 500ml soft flasks for long runs and trail outings. Minimal bounce design with adjustable sternum straps.

## Best For
- Long-distance trail and road running
- Ultra and marathon training
- Hiking and fastpacking

## Key Features
- Two 500ml soft flasks included (front chest pockets)
- Rear 1.5L reservoir sleeve (reservoir sold separately)
- Multiple zippered and stretch pockets for nutrition and gear
- Dual adjustable sternum straps for bounce-free fit
- Reflective accents
- Weight: 7.2 oz (without flasks)

## Sizing
- Available: XS/S, M/L, XL/XXL
- Adjustable fit via sternum straps and elastic side panels

## Materials
- Body: Nylon ripstop mesh
- Flasks: BPA-free TPU
- Hand wash flasks after each use; vest spot clean
