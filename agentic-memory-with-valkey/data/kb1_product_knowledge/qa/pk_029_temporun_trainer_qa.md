# TempoRun Interval Trainer Community Q&A (Sample)

## Q1: How does this compare to SprintEdge Racer?
**Answer (Brand Rep):** TempoRun has more cushioning and no carbon plate, making it more versatile for speed workouts and tempo runs. SprintEdge Racer is lighter and faster but designed only for race day. TempoRun bridges the gap between daily trainer and race flat.

## Q2: Can I use this as my only running shoe?
**Answer (Community):** You could, but it's best as a speed day shoe. The thin outsole wears faster than a daily trainer. I rotate between AeroFlex Runner 2 for easy days and TempoRun for workouts.

## Q3: Does the narrow heel cause blisters?
**Answer (Community):** It did for my first two runs. After that it broke in and was fine. Try a heel-lock lacing technique if it bothers you.

## Q4: Is there a wide version?
**Answer (Brand Rep):** TempoRun Interval Trainer is currently available in standard width only.

## Q5: What's the drop (heel-to-toe offset)?
**Answer (Brand Rep):** TempoRun has a 6mm heel-to-toe drop, compared to 8mm for AeroFlex Runner 2 and 4mm for SprintEdge Racer.
