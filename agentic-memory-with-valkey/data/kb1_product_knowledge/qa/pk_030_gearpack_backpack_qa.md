# GearPack Running Backpack 15L Community Q&A (Sample)

## Q1: Can I fit a laptop in this?
**Answer (Brand Rep):** GearPack does not have a dedicated laptop sleeve. The main compartment can physically fit a thin 13" laptop but there is no padding or protection for electronics. This is designed as a running pack, not a commuter bag.

## Q2: Does it have a hydration bladder sleeve?
**Answer (Brand Rep):** Yes. There is an internal hydration reservoir sleeve that fits up to a 2L bladder. Reservoir is sold separately.

## Q3: Does it bounce when running?
**Answer (Community):** With the compression straps tightened and sternum strap clipped, bounce is minimal. I run 7-minute miles with a full change of clothes and it stays put.

## Q4: Is it carry-on size for flights?
**Answer (Community):** At 15L it's well under any carry-on limit. I use it as my personal item on flights.

## Q5: Can I remove the hip belt?
**Answer (Brand Rep):** Yes. The hip belt is removable. Most runners find the sternum strap alone provides sufficient stability.
