# FitTech Solutions: SwiftPace Watch Shipped Without Charging Cable

## Summary
FitTech Solutions (FTS-US-502) shipped approximately 200 SwiftPace Running Watch orders in February 2026 without the magnetic charging cable included. The watch, quick start guide, and box were present — only the cable was missing.

## Timeline
- Feb 8–14: 23 customer complaints about missing cable
- Feb 15: Pattern identified by Marketplace Quality team
- Feb 16: FitTech notified. Root cause: warehouse packing station reorganization caused cable bin to be moved, packers missed it.
- Feb 17: FitTech began proactively shipping cables to all orders from Feb 8–14 batch
- Feb 20: Packing process corrected. No further reports.

## Current Status
- FitTech is proactively shipping replacement cables. Most customers should receive within 3–5 business days of Feb 17.
- Some customers may contact before receiving the proactive shipment.

## Agent Action
If customer reports missing SwiftPace charging cable and purchased from FitTech in Feb 2026:
1. Check if order date is Feb 8–14. If yes: "FitTech has already identified this issue and shipped a replacement cable to you. It should arrive within a few business days."
2. If customer hasn't received replacement by Feb 24: escalate to FitTech seller support for tracking.
3. If order date is outside Feb 8–14: may be a different issue. Verify with FitTech.
4. **In the meantime:** SwiftPace charges via any Qi-compatible magnetic charger. If customer has one, they can use it temporarily. The OEM cable is magnetic USB-C — not proprietary.

## Cross-Reference
This is a **seller fulfillment issue**, not a product defect. Do not confuse with SwiftPace firmware bricking issue (KB3: ts_022) which is a completely separate problem.
