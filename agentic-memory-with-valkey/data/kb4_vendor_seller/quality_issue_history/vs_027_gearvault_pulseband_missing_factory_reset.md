# GearVault Outlet: Renewed PulseFit Band 4 Units Missing Firmware Reset

## Summary
GearVault (GVO-US-618) sold approximately 150 renewed PulseFit Band 4 units in Jan–Feb 2026 that were not factory-reset before resale. Customers received devices with previous owner's data (activity history, sleep data, paired phone info) still on the device.

## Timeline
- Jan 20 – Feb 15: 18 customer complaints about pre-existing data on "new" PulseFit Band 4
- Feb 16: Pattern identified by Marketplace Quality team
- Feb 17: GearVault notified. Root cause: refurbishment process skipped factory reset step for a batch of ~150 units.
- Feb 18: GearVault updated refurbishment checklist. Remaining unsold units from batch being re-processed.

## Privacy Concern
This is a **data privacy issue**. Previous owner's health data (heart rate, sleep patterns, activity) is visible to the new owner. While the data is anonymized (no name or account info on the device itself), it's still a privacy concern.

## Agent Action
If customer reports pre-existing data on a renewed PulseFit Band 4 from GearVault:
1. Apologize: "I'm sorry about that. The device should have been factory-reset before resale."
2. Walk customer through factory reset: Settings → System → Reset → Factory Reset. This erases all previous data.
3. After reset, customer sets up as new device with their own account.
4. Reassure: "The previous owner's account information was not stored on the device — only activity data, which has now been erased."
5. If customer is uncomfortable and wants to return: process through GearVault's 30-day return policy.
