# RunReady Authorized: AeroFlex Runner 2 Sizing Complaint Surge

## Summary
RunReady (RRA-US-415) experienced a 3x increase in sizing-related returns for AeroFlex Runner 2 in January 2026. Investigation revealed RunReady's product listing did not include the "runs half size small" advisory that appears on the official Accent Athletics product page.

## Timeline
- Jan 5–25, 2026: 47 sizing returns (vs typical 15/month)
- Jan 26: Accent Athletics Marketplace team flagged the trend
- Jan 28: RunReady updated listing to include sizing advisory
- Feb 5: Return rate returning to normal

## Root Cause
RunReady copied the AeroFlex Runner 2 product description but omitted the sizing callout box that says "This shoe runs approximately half a size small. We recommend ordering a half size up from your usual size." This callout is prominently featured in the official product description (KB1: pk_001) and fit guide (KB1: pk_005).

## Current Status
- Listing has been corrected
- RunReady issued prepaid return labels to all affected customers
- No further action needed from Accent Athletics support

## Agent Relevance
If a customer contacts about AeroFlex Runner 2 sizing and purchased from RunReady in January 2026:
- The sizing issue is real (shoe does run small — see pk_005)
- RunReady has already been notified and will accept the return/exchange
- Process through RunReady's exchange flow (vs_017)
- Do NOT blame RunReady to the customer — just help them get the right size
