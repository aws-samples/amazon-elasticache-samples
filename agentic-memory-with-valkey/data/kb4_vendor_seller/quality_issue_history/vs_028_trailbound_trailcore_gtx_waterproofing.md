# TrailBound Outfitters: TrailCore Grip 5 GTX Waterproofing Complaints

## Summary
TrailBound (TBO-US-833) received 8 complaints in Feb 2026 about TrailCore Grip 5 GTX waterproofing failure. Customers report water leaking through the GTX membrane during stream crossings and heavy rain.

## Investigation
- Accent Athletics Quality team tested 5 units from TrailBound's inventory batch
- 2 of 5 units showed compromised GTX membrane seam tape (delaminating at the toe box)
- Remaining 3 units passed waterproof testing
- Batch affected: manufactured Nov 2025, batch code GTX-2511-B

## Root Cause
Manufacturing defect in batch GTX-2511-B. Seam tape adhesive was applied at incorrect temperature during production, causing premature delamination under stress (flexing during trail running).

## Scope
- This is a **manufacturer defect**, not a TrailBound issue. TrailBound's storage and handling were verified as correct.
- Batch GTX-2511-B was distributed to TrailBound and direct Accent Athletics inventory. Both channels are affected.
- Estimated 400 pairs in batch. Not all will fail — defect rate appears ~40% based on sample.

## Agent Action
If customer reports TrailCore Grip 5 GTX waterproofing failure:
1. Check batch code (printed inside tongue, format: GTX-YYMM-X). If GTX-2511-B: confirmed affected batch.
2. Process **manufacturer warranty replacement** regardless of purchase source (TrailBound or direct).
3. If batch code is different: standard waterproofing troubleshooting. GTX membrane can be compromised by machine washing, dryer heat, or prolonged exposure to solvents. Ask about care history.
4. Replacement ships from Accent Athletics directly, not TrailBound.
