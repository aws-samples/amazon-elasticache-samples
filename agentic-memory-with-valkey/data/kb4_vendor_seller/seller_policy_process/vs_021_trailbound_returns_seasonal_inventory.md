# TrailBound Outfitters Returns & Seasonal Inventory Process

## Purpose
Defines return handling and seasonal inventory notes for trail/outdoor products purchased from TrailBound Outfitters (TBO-US-833).

## Return Policy
- **45-day return window** for unworn items in original packaging
- **90-day exchange window** for sizing exchanges on footwear
- Used/worn items: TrailBound accepts returns on trail shoes worn fewer than 30 miles if customer is unsatisfied with fit on actual trails. This is a TrailBound-specific policy — more generous than Accent Athletics direct (which requires unworn).

## Trail Shoe Sizing Notes
TrailBound's #1 return reason is trail shoe sizing. Cross-reference with KB1 fit guide (pk_005):
- TrailCore Grip 5: recommend half size up if customer plans to wear thick hiking socks or use aftermarket insoles
- TrailCore Grip 5 GTX: GTX lining makes the shoe slightly snugger than non-GTX. Recommend half size up from non-GTX size.
- **TrailCore Grip 4 is discontinued** (KB1: pk_032 deprecated). If customer asks TrailBound about Grip 4: it's no longer available. Recommend Grip 5 as replacement per substitution playbook (KB1: pk_006).

## Seasonal Inventory Gaps
TrailBound is a smaller seller and may run out of popular sizes during peak seasons:
- **Spring (Mar–May):** TrailCore Grip 5 sizes 10–11 frequently out of stock. If customer can't get their size from TrailBound, direct to accentathletics.com.
- **Fall (Sep–Nov):** ThermoShield Running Jacket sizes M–L sell out fast. Same redirect.
- TrailBound does NOT do backorders. If out of stock, it's out of stock.

## Warranty Routing
Same as RunReady: full Accent Athletics manufacturer warranty. Physical defects → Accent Athletics warranty team. Sizing/preference returns → TrailBound.
