# FlexWear Collective Marketplace Fulfillment Ownership Guide

## Purpose
Help agents determine whether FlexWear Collective orders follow retailer return flow or seller return flow.

## Key Rule
FlexWear Collective operates a mixed fulfillment model. The same seller may have:
- Retailer-fulfilled listings
- Seller-fulfilled listings

Support instructions depend on the fulfillment tag on the order/SKU.

## Decision Flow
1. Confirm order line item seller = FlexWear Collective
2. Check fulfillment tag on order line:
   - If **Retailer-Fulfilled** -> follow retailer standard shipping/returns workflow
   - If **Seller-Fulfilled** -> follow FlexWear Collective seller process
3. Check SKU notes for fragile-item handling requirements

## Fragile Item Claims (Seller-Fulfilled)
For fragile fitness equipment:
- Request photos of item and external packaging
- Record visible damage at corners/edges if present
- Escalate repeat damage complaints on same SKU to quality tracking

## Common Agent Mistake to Avoid
Do not assume all FlexWear Collective orders can be returned in store. Eligibility depends on fulfillment ownership.
