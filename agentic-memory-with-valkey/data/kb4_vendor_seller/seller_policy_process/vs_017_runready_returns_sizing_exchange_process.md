# RunReady Authorized Returns & Sizing Exchange Process

## Purpose
Defines return and exchange handling for Accent Athletics footwear and apparel purchased from RunReady Authorized (RRA-US-415).

## Return Window
- **30 days** from delivery for unworn items in original packaging
- **60 days** for sizing exchanges only (customer gets different size of same product)
- No returns on final-sale items (marked on listing)

## Sizing Exchange Process
RunReady is the #1 source of sizing-related exchanges on the marketplace. Common scenario: customer orders AeroFlex Runner 2 based on their usual size, but Accent Athletics sizing runs differently.

**Agent action for sizing complaints:**
1. First, consult the Accent Athletics Fit & Sizing Guide (KB1: pk_005) to verify if the customer's experience matches known sizing patterns.
2. If the guide confirms the product runs large/small/narrow: validate the customer's experience and process exchange to recommended size.
3. If within 60-day exchange window: initiate exchange through RunReady's seller portal. Customer ships back to RunReady (prepaid label provided).
4. If outside 60-day window: RunReady will not accept. Check if Accent Athletics direct return policy applies (it doesn't — marketplace purchases follow seller policy).

## Cross-Reference: Known Sizing Issues
Per KB1 fit guide (pk_005):
- AeroFlex Runner 2 runs **half size small** — most common exchange reason
- StrideMax Cushion Pro runs **true to size** but narrow in the toe box
- TrailCore Grip 5 runs **true to size** but recommend half size up for thick hiking socks
- SprintEdge Racer is a **race fit** — intentionally snug, not a defect

## Defective Product Process
- If product is defective (sole separation, stitching failure, etc.): this is a **manufacturer warranty issue**, not a RunReady return.
- Route to Accent Athletics warranty team. Do NOT send customer back to RunReady for defects.
- RunReady is not responsible for manufacturing defects.
