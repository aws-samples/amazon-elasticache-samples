# Marketplace Authorized Seller Product Authenticity Verification Guide

## Purpose
Help agents verify whether a product is genuine Accent Athletics when a customer questions authenticity, especially after the PeakForm suspension (vs_015).

## When to Use This Guide
- Customer says "I think my product might be fake"
- Customer purchased from a third-party seller (not direct from Accent Athletics)
- Product behavior doesn't match official documentation (KB1 product descriptions, KB3 troubleshooting)

## Verification Methods by Product Type

### Footwear (AeroFlex, TrailCore, StrideMax, etc.)
1. **Size tag:** Inside tongue should show "Accent Athletics" logo, size, style code (format: AA-XXXX-XXX), and country of manufacture.
2. **Style code check:** Customer provides style code → agent verifies against product catalog. If code doesn't match any known product, likely counterfeit.
3. **Box label:** UPC barcode on box should match product listing UPC. QR code on box should link to accentathletics.com/verify.
4. **Construction quality:** Genuine AeroFlex Runner 2 has dual-density midsole with visible color separation. Counterfeits often have single-density painted to look dual.

### Electronics (SwiftPace, AeroTrack, PulseFit Band, StrideScale, SleepSync)
1. **Serial number:** Settings → About → Serial Number. Format: [Product Code][Year Quarter]-[5 digits]. Example: SP25Q3-12345 for SwiftPace manufactured Q3 2025.
2. **Firmware check:** Genuine devices connect to Accent Athletics update servers. If device cannot check for updates or shows "Unknown device" in app, it's not genuine.
3. **App pairing:** Genuine devices pair with official Accent Athletics apps (Health or Audio). Counterfeits will not be recognized by the app.
4. **Charging accessories:** Genuine PowerBase Charging Dock has "Accent Athletics" engraved (not printed) on the bottom.

### Apparel
1. **Care label:** Should show Accent Athletics logo, fabric composition, and style code.
2. **Hang tag:** Genuine products have a holographic hang tag with QR code linking to accentathletics.com/verify.
3. **Fabric quality:** Genuine DryStride Performance Tee uses proprietary DryStride fabric with a distinctive texture. Counterfeits often use standard polyester.

## If Product Is Confirmed Counterfeit
Follow vs_020 (PeakForm Counterfeit Handling Playbook) process — it applies to any counterfeit regardless of seller.

## If Product Is Confirmed Genuine
Proceed with normal support. The customer's concern was unfounded — reassure them and address their original issue.

## If Verification Is Inconclusive
Escalate to Brand Protection team via ticket (category: BRAND-AUTHENTICITY-CHECK). Include photos if customer can provide. Response SLA: 48 hours.
