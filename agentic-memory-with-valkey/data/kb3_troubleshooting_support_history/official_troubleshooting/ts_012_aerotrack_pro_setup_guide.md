# AeroTrack Pro Earbuds Initial Setup Guide

## Purpose
Walk customers through first-time setup of AeroTrack Pro wireless earbuds, including Bluetooth pairing, companion app installation, and ear tip fitting.

## Box Contents
- AeroTrack Pro earbuds (L/R)
- Charging case
- USB-C charging cable (30 cm)
- Ear tips: S, M (pre-installed), L, XL
- Quick start card

## Step 1: Charge Before First Use
Place earbuds in case, connect USB-C cable. LED on case front:
- Red = charging
- Green = fully charged (~90 min from empty)
- Amber = 50–80%

Do NOT attempt pairing until case LED shows green or amber. Pairing on low battery causes intermittent disconnects.

## Step 2: Download Companion App
- iOS: App Store → search "Accent Athletics Audio"
- Android: Google Play → search "Accent Athletics Audio"
- Minimum OS: iOS 16+ / Android 12+

## Step 3: Bluetooth Pairing
1. Open case lid with earbuds inside. Case LED blinks white = pairing mode.
2. On phone: Settings → Bluetooth → look for "AeroTrack Pro - XXXX" (last 4 of serial).
3. Tap to pair. Accept pairing request.
4. Open companion app → it should auto-detect paired earbuds.

**If earbuds don't appear in Bluetooth list:** Close case lid, wait 5 seconds, reopen. If still not visible, press and hold case button for 8 seconds until LED blinks red/white alternating (factory reset), then retry.

## Step 4: Ear Tip Fitting
- Open companion app → Settings → Ear Tip Fit Test
- App plays test tone and measures seal quality
- "Good Seal" = correct size. "Poor Seal" = try next size up
- Proper seal is critical for ANC (Active Noise Cancellation) performance

## Step 5: Firmware Update
- App will prompt for firmware update on first connection. **Always accept.** Updates fix bugs and improve ANC algorithms.
- Update takes ~5 min. Do not remove earbuds from case during update.
- Current firmware: v2.4.0 (as of Feb 2026)

## Common First-Setup Issues
| Issue | Fix |
|---|---|
| Only one earbud pairs | Place both in case, close lid 10 sec, reopen. If persists, factory reset. |
| App doesn't detect earbuds | Force-close app, re-open. Ensure Bluetooth is on and earbuds show as connected in phone settings. |
| Ear tip test always fails | Try XL tips. Some ear canals need the largest size. If all fail, earbuds may be inserted upside down — check L/R markings. |
