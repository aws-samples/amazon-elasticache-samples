# SleepSync Tracker Setup and Troubleshooting Guide

## Purpose
Combined setup and troubleshooting guide for the SleepSync athletic recovery sleep tracker. Covers initial setup, common issues, and sensor calibration.

## Product Overview
SleepSync is a non-contact athletic recovery tracker that sits on the nightstand. Uses radar-based sensing to monitor breathing rate, movement, and sleep stages without wearing anything. Syncs to Accent Athletics Health app via Wi-Fi.

## Setup

### Placement
- Place on nightstand **within 2 feet of pillow**, at mattress height or slightly above.
- Sensor faces the bed. Arrow on bottom of device should point toward sleeper's chest.
- Do NOT place behind objects (lamp, books) — radar needs clear line of sight.
- For couples: one SleepSync per side of bed. Each tracks the nearest sleeper.

### Power & Connection
- USB-C power (must be plugged in — no battery). Use included 6-ft cable + adapter.
- Wi-Fi setup: same as StrideScale — 2.4 GHz only, via Accent Athletics Health app.
- LED meanings: solid white = ready, breathing blue = tracking active, red = error.

### First Night Calibration
- SleepSync needs 2–3 nights to calibrate to the sleeper's breathing pattern.
- First night data may show gaps or "Unknown" sleep stages. This is normal.
- By night 3, sleep stage detection (Light / Deep / REM) should be accurate.

## Common Issues

### "No Sleeper Detected"
- Check placement: within 2 feet, arrow pointing at chest, no obstructions.
- If sleeper is a very still sleeper: increase sensitivity in app (Devices → SleepSync → Sensitivity → High).
- Thick/heavy comforters can attenuate radar signal. Try with thinner blanket as test.
- Pets: if a large dog sleeps on the bed, SleepSync may track the dog instead. Reposition to aim specifically at human.

### Inaccurate Sleep Stages
- Normal for first 2–3 nights (calibration period).
- After calibration: check firmware. v1.3.0 significantly improved REM detection. Update if on older version.
- Room temperature extremes (below 60°F or above 80°F) can affect breathing patterns and confuse the algorithm. Recommend 65–72°F.
- Alcohol consumption significantly alters breathing patterns — sleep data on those nights will be less accurate. This is expected, not a device issue.

### Tracks Partner Instead of User
- Reposition device closer to correct sleeper.
- If nightstand is between both sleepers: mount on wall behind headboard using included adhesive mount, aimed at correct side.
- App → Devices → SleepSync → Tracking Zone → adjust radar cone angle (narrow vs wide).

### Wi-Fi Sync Issues
- Same troubleshooting as StrideScale (see ts_015). 2.4 GHz only, check signal strength.
- SleepSync syncs data in the morning after tracking ends. If user wakes and immediately checks app, data may not be available for 5–10 minutes.

## Escalation
Escalate to Tier 2 if: "No Sleeper Detected" persists after placement adjustment + sensitivity increase + firmware update. Include photo of device placement if possible.
