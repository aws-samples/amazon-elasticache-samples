# StrideScale Smart Body Composition Scale Setup Guide

## Purpose
Guide customers through initial setup of the StrideScale smart body composition scale for athletes, including Wi-Fi connection, user profile creation, and first measurement.

## Box Contents
- StrideScale scale unit
- 4x AAA batteries (pre-installed)
- Quick start card

## Step 1: Place on Hard, Flat Surface
- **Must be on hard floor** (tile, hardwood, concrete). Carpet causes inaccurate readings.
- If customer only has carpet: recommend purchasing hard floor pad accessory ($12.99 on website).
- Level surface required. Scale has auto-level detection — if surface is uneven, app will show "Uneven Surface" warning.

## Step 2: Download App & Create Profile
- iOS/Android: search "Accent Athletics Health"
- Create account or sign in with existing Accent Athletics account
- Add new device → select "StrideScale" → follow on-screen prompts
- Enter profile: height, age, gender (used for body composition algorithms)

## Step 3: Wi-Fi Connection
- Scale connects to 2.4 GHz Wi-Fi only. **5 GHz networks are not supported.**
- In app: Device Setup → Wi-Fi → select network → enter password
- Scale LED blinks blue during connection, solid blue = connected
- If connection fails: verify 2.4 GHz band. Many modern routers combine bands — customer may need to split them or use the 2.4 GHz SSID specifically.

## Step 4: First Measurement
- Step on scale barefoot (bioimpedance sensors require skin contact)
- Stand still for ~8 seconds until scale beeps
- App displays: weight, BMI, body fat %, muscle mass, bone mass, water %
- First reading may be less accurate for body composition — scale calibrates over 3–5 readings

## Common Setup Issues
| Issue | Fix |
|---|---|
| Scale won't turn on | Remove battery cover, reseat batteries. Check polarity. |
| Wi-Fi connection fails | Confirm 2.4 GHz. Move scale closer to router for setup, then move to final location. |
| App doesn't find scale | Ensure Bluetooth is enabled on phone (used for initial setup even though scale uses Wi-Fi for data sync). |
| "Uneven Surface" warning | Move to different spot. If persistent on hard floor, check for debris under scale feet. |
| Body comp readings seem wrong | Normal for first 3–5 uses. Measure at same time daily (morning, before eating) for consistency. |
