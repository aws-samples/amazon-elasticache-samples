# Legacy SwiftPace GPS Calibration Procedure (Deprecated)
**Status:** Deprecated

> **Status: DEPRECATED** | Last updated: 2025-10-01
> **Superseded by:** ts_017 (SwiftPace GPS Troubleshooting — current)

## Original Content (Do Not Use)
This document described a manual GPS calibration procedure for SwiftPace firmware v2.x:
1. Open Settings → Sensors → GPS → Calibrate
2. Walk in a figure-8 pattern for 30 seconds
3. Watch would display "Calibration Complete"

## Why Deprecated
The manual calibration feature was **removed in firmware v3.0.0**. GPS calibration is now fully automatic. The figure-8 procedure does nothing on v3.x firmware — the menu option no longer exists.

**Common mistake:** Some agents and online forums still recommend the figure-8 calibration. If a customer mentions it, explain: "That calibration method was for an older firmware version. Your current firmware handles GPS calibration automatically. Let me walk you through the current troubleshooting steps instead."
