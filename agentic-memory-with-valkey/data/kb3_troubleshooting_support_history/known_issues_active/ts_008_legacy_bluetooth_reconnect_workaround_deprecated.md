# Legacy Bluetooth Reconnect Workaround (Deprecated)

> **Status: DEPRECATED** | Last updated: 2026-01-21
> This workaround was used for the resolved 4.1.x reconnect loop issue (ts_007) and should not be used for current sync delay cases on firmware 4.2.1. See ts_004 for the active workaround.

Deprecated sequence (no longer recommended):
- Forget device from phone Bluetooth settings
- Toggle airplane mode twice
- Reinstall app
