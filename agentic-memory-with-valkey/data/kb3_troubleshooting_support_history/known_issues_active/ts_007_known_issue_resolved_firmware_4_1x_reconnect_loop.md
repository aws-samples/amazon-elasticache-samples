# Known Issue Resolved: Firmware 4.1.x Bluetooth Reconnect Loop
**Status:** Resolved  
**Resolution Date:** 2026-01-20

## Summary
A subset of PulseFit Band 4 devices on firmware 4.1.x experienced repeated Bluetooth reconnect loops after app background resume. This issue was resolved in firmware 4.2.0.

## Agent Guidance
- If customer is on firmware 4.1.x, recommend updating to latest firmware
- Do not use legacy reconnect workaround if device is already on 4.2.0 or later
