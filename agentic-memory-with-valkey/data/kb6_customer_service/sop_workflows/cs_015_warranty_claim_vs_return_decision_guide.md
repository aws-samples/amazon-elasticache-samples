# SOP: Warranty Claim vs. Return — Decision Guide

## Purpose
Help agents distinguish between warranty claims and returns, which follow different processes and policies.

## The Key Question
**Is the customer unhappy with the product, or is the product defective?**

| Customer Says | Classification | Process |
|---|---|---|
| "I don't like it" / "It's not what I expected" | Return | pc_002 returns policy |
| "It doesn't fit" / "Wrong size" | Return (sizing) | pc_002 + KB1: pk_005 fit guide |
| "It broke" / "It stopped working" | Warranty claim | pc_011 (electronics) or pc_012 (footwear/apparel) |
| "It's defective" / "Manufacturing issue" | Warranty claim | pc_011 or pc_012 |
| "The firmware update broke it" | Warranty claim (manufacturer fault) | pc_020 (SwiftPace bricking exception) |
| "It's fake" / "Not authentic" | Counterfeit | KB4: vs_020 + pc_021 |

## Return Process Summary
1. Check return window: 60 days unopened, 30 days opened (pc_002). In-store: see pc_014.
2. Check purchase source: direct → Accent Athletics handles. Marketplace seller → seller handles (KB4).
3. Check for active exceptions: holiday extension (pc_006), weather delay (pc_007).
4. Process return or redirect to seller.

## Warranty Claim Process Summary
1. Troubleshoot first (KB3 for electronics, visual inspection for footwear/apparel).
2. If defect confirmed: initiate warranty replacement.
3. Check purchase source for warranty eligibility:
   - Direct or authorized seller (RunReady, FitTech, TrailBound): 1-year manufacturer warranty
   - GearVault: 90-day GearVault warranty for hardware, full Accent Athletics support for software
   - PeakForm: NO warranty (counterfeit)
4. Check for active warranty exceptions: SwiftPace bricking (pc_020) overrides normal warranty.
5. If out of warranty: offer repair service (electronics) or suggest replacement with substitution guidance (KB1: pk_006).

## Common Mistakes
1. ❌ Treating a sizing complaint as a defect → wastes warranty team's time
2. ❌ Treating a firmware bug as a return → customer shouldn't have to return a product that Accent Athletics broke with an update
3. ❌ Telling marketplace customers "contact the seller" for defects → manufacturer warranty is Accent Athletics' responsibility (pc_028 deprecated this old approach)
4. ❌ Applying the old 14-day electronics return window → deprecated (pc_027). Current window is 30 days opened.
5. ❌ Offering only $10 goodwill credit → deprecated cap (pc_029). Current Tier 1 limit is $15, higher with active exceptions.
