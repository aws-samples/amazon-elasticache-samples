# Announcement: Updated Marketplace Ownership Check Before Return Guidance
**Status:** Active  
**Effective Date:** 2026-02-16

## Summary
Support teams must confirm fulfillment ownership (retailer-fulfilled vs seller-fulfilled) before giving return instructions on marketplace orders.

## Why This Changed
Recent QA reviews found avoidable errors where agents quoted retailer in-store return guidance for seller-fulfilled marketplace orders.

## What Changes for Agents
Before discussing return eligibility or process for marketplace orders:
1. Confirm seller and fulfillment ownership on the order line
2. Check seller-specific policy/process guidance when seller-fulfilled
3. Use marketplace policy override reference if applicability is unclear
4. Use marketplace macro wording where appropriate

## Impact
This change reduces incorrect return promises and seller-policy conflicts.
