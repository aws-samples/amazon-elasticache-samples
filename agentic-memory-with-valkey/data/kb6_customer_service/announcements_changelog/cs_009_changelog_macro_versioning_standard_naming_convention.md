# Change Log: Macro Versioning Standard and Naming Convention
**Status:** Approved  
**Effective Date:** 2026-02-01

## Summary
Macro packs now use standardized IDs and version labels to reduce confusion across teams.

## New Standard
- Macro pack version format: `MAC-<DOMAIN>-<REGION>-v<major.minor>`
- Macro ID format: `<DOMAIN>_<USECASE>_<NN>`

## Agent Impact
- Always use the latest approved macro pack
- Do not save personal copies of macros outside approved library
- Deprecated macros may remain searchable for audit purposes but should not be used in active customer replies
