# US In-Store Return Policy Addendum
**Policy Version:** RET-STORE-US-v2.0
**Region:** US
**Status:** Approved
**Effective:** 2026-01-01 to 2026-12-31
**Supplements:** RET-US-v4.1 (pc_002)

## Purpose
Clarifies return policy specifics for in-store transactions. This addendum supplements the main returns policy (pc_002) — it does not replace it.

## In-Store Return Windows
| Item Type | Return Window | Condition |
|---|---|---|
| Footwear (unworn) | 60 days | Original box, tags attached |
| Footwear (worn, defective) | 1 year (warranty) | Photo evidence of defect |
| Apparel (unworn) | 60 days | Tags attached |
| Apparel (worn, defective) | 6 months (warranty) | Visible defect |
| Electronics (unopened) | 60 days | Sealed packaging |
| Electronics (opened, non-defective) | 30 days | All accessories included, factory reset |
| Electronics (defective) | 1 year (warranty) | See pc_011 |
| Accessories (socks, gloves, hats) | 30 days | Unworn, tags attached |
| Final sale items | No returns | Clearly marked at purchase |

## Marketplace Seller Products Cannot Be Returned In-Store
- Products purchased from RunReady, FitTech, TrailBound, GearVault, or any marketplace seller **cannot be returned at Accent Athletics stores**.
- If customer brings a marketplace purchase to a store: politely redirect to the seller's return process.
- Common scenario: customer bought AeroFlex Runner 2 from RunReady, wants to return at Chicago Michigan Ave store (KB2: so_015). Store cannot accept. Direct to RunReady return process (KB4: vs_017).

## Cross-Store Returns
- Customers may return at ANY Accent Athletics store, regardless of which store they purchased from.
- Cross-metro returns accepted (e.g., bought in NYC, return in LA). Use inter-store return code in POS.
- See KB2 store notes for store-specific return handling (e.g., so_014 Beverly Center accepts returns from other LA stores).

## Receipt Requirements
- With receipt/order confirmation: full refund to original payment method
- Without receipt: store credit only, at current selling price (may be lower than purchase price if item is now on sale)
- Digital receipt lookup: agent can search by customer email, phone, or loyalty account

## Holiday Exception
During active holiday return extension (pc_006): in-store return window extends to Jan 31 for purchases made Nov 15–Dec 31. This applies to 2026 holiday season. Do NOT apply the expired 2025 extension (pc_008).
