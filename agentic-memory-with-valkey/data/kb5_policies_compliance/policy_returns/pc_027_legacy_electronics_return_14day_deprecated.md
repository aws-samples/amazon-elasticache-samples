# Legacy Electronics Return Policy — 14-Day Window (Deprecated)
**Policy Version:** RET-ELEC-US-v1.0
**Status:** Deprecated

> **Status: DEPRECATED** | Last updated: 2025-12-31
> **Superseded by:** RET-US-v4.1 (pc_002) which standardized all return windows to 60 days unopened / 30 days opened.

## Original Content (Do Not Use)
Prior to 2026, Accent Athletics had a separate electronics return policy with a **14-day return window** for opened electronics. This was more restrictive than the general return policy.

The 14-day window was eliminated on Jan 1, 2026 when the returns policy was unified. All electronics now follow the standard 30-day opened / 60-day unopened window per pc_002.

## Why Deprecated
Customer feedback consistently cited the 14-day electronics window as too short, especially for devices that require setup and calibration (SleepSync needs 2–3 nights to calibrate, StrideScale needs 3–5 readings). By the time customers could evaluate the product, the return window was nearly closed.

## Common Mistake
Agents trained before 2026 sometimes still quote the "14-day electronics return window." This is WRONG. The current window is 30 days opened / 60 days unopened. If an agent tells a customer they're outside the 14-day window, they are providing incorrect information and potentially denying a valid return.
