# Temporary Holiday Return Extension (US)
**Exception ID:** TEMP-RET-US-HOL-2026  
**Region:** US  
**Status:** Active (Seasonal)  
**Effective:** 2026-11-15 through 2026-12-31

## Purpose
Provide extended return eligibility for qualifying holiday purchases made during the seasonal promotion period.

## Eligibility Summary
Eligible items purchased during the holiday program window may be returned through **January 31, 2027**, unless excluded by category, seller policy, or law.

## Exclusions (Examples)
- Final sale items
- Certain hygiene-sensitive products once opened
- Personalized/custom items
- Seller-fulfilled marketplace items where seller policy differs (unless marketplace protections apply)

## Agent Guidance
- Confirm purchase date and item category before quoting holiday extension
- Confirm fulfillment ownership for marketplace orders
- If item is excluded, use approved return denial or seller-policy difference messaging

## Override Effect
This exception overrides the standard 30-day return window in the US Returns, Refunds, and Exchanges Policy (RET-US-v4.1) for in-scope purchases only.
