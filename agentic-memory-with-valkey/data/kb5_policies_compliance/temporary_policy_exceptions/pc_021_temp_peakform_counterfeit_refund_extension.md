# Temporary Policy Exception: PeakForm Counterfeit Refund Extension
**Exception ID:** TEMP-REF-PEAKFORM-2026-02
**Region:** US
**Status:** Active
**Effective:** 2026-02-15 to 2026-05-15
**Overrides:** RET-US-v4.1 (pc_002)
**Exception Priority:** 1

## Exception
Due to the PeakForm Athletics suspension for counterfeit products (KB4: vs_015), the standard 60-day return window is extended:

**Any product purchased from PeakForm Athletics is eligible for full refund regardless of:**
- Purchase date (even if outside standard 60-day return window)
- Product condition (used, worn, opened — doesn't matter, it's counterfeit)
- Whether customer has the product (refund even if discarded)

## Refund Window
- Orders within last 90 days: automatic refund via Marketplace Refund Tool (reason: "Seller Suspended — Counterfeit")
- Orders 90–180 days ago: refund approved, process manually via Support Refund Tool
- Orders >180 days ago: escalate to Marketplace team for case-by-case review

## Process
1. Verify order was from PeakForm (seller ID PFA-US-720)
2. Process refund per above windows
3. Follow KB4: vs_020 counterfeit handling playbook for customer communication
4. If electronics product: provide safety advisory per vs_020 and pc_017
5. Do NOT ask customer to return the product (counterfeit goods should not re-enter any supply chain)

## Budget
Refunds funded by Marketplace Seller Trust & Safety budget, not customer service budget. No impact on agent refund metrics.
