# Bellevue Square Mall Access and Pickup Instructions
**Store:** BEL-014 (Bellevue Square)  
**Region:** US-WA  
**Status:** Active

## Customer-Facing Service Note
BEL-014 supports in-store pickup during store operating hours. Curbside pickup is not available at this location due to mall property access restrictions.

## Pickup Instructions
- Proceed to the main store entrance on Level 2
- Bring order confirmation and valid ID
- During high traffic periods, check in at the service desk queue sign

## Agent Guidance
- Do not offer curbside for BEL-014
- If customer requests curbside due to mobility constraints, escalate to store leadership for case-by-case accommodation if local process permits
- Offer alternate nearby store with curbside only if customer asks for options

## Mall Access Notes
Holiday and event traffic may increase pickup wait times. Use current store operating status tools before quoting wait expectations.
