# Dallas NorthPark Center Store Note
**Store:** DFW-001 (NorthPark Center)
**Region:** US-TX
**Status:** Active
**Effective:** 2026-01-01 to 2026-12-31

## Store Details
- Premium mall location between Neiman Marcus and Nordstrom. ~3,800 sq ft.
- Pickup inside store only (mall policy, no curbside).
- Two fitting rooms, one ADA-compliant.

## Special Procedures
- **Tornado season (Mar–Jun):** If NWS issues tornado warning for Dallas County, follow mall shelter-in-place protocol. Move customers to interior hallway away from glass. Do NOT attempt to close/lock store during active warning.
- **State Fair of Texas (Oct):** Fair Park is 5 miles south. Moderate traffic increase. Stock Texas-themed accessories if available.
- **Returns from Fort Worth stores:** Accept cross-metro returns. Use inter-store return code DFW-XFER in POS.

## Parking
- NorthPark garage: free, covered. Direct to Level 2 near Nordstrom entrance for closest access.
